按钮[button]

自定义的按钮控件,灵活定制,不拘一格 按钮网页控件
按钮
常规按钮 中号按钮 大号按钮 常规按钮 中号按钮 大号按钮 删除 禁用
如何使用

引入CSS

<style type="text/css" src="/media/css/elfWidgetDefine.css"></style><!--控件定义-->
<style type="text/css" src="/media/css/elfWidgetColor.css"></style><!--控件配色-->

-html方式创建按钮:

<a href="#" class="button"><span>常规按钮</span></a>
      <a href="#" class="button"><span class="M">中号按钮</span></a>
      <a href="#" class="button"><span class="L">大号按钮</span></a>
      <a href="#" class="button"><span class="contrast">常规按钮</span></a>
      <a href="#" class="button"><span class="contrast M">中号按钮</span></a>
      <a href="#" class="button"><span class="L contrast">大号按钮</span></a>
      <a href="#" class="button"><span class="M careful">删除</span></a>
      <a href="#" class="button disable"><span class="M">禁用</span></a>

-javascript方式创建按钮:

//...(敬请关注)
注意事项

1.M为中号,L为大号,默认为小号.

2.您可以根据项目不同修改配色文件elfWidgetColor.css
